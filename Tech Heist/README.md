# UniNotes – Academic Resource Hub

## Overview

UniNotes is a simple application that helps students and teachers manage all study materials in one place.
Teachers can add study materials like notes and assignments, and students can access all of it in one place in an organized manner.

## Problem

Study materials like notes and assignments are shared through WhatsApp groups or via email in most colleges.
This leads to students not finding the relevant study materials or not having access to all study materials.

## Solution

UniNotes is a simple application where teachers can add all study materials in one place, and students can access all of it in one place in an organized manner.

## Features

* Teacher Dashboard: where teachers can add study materials like notes and assignments
* Student Dashboard: where students can access all study materials in one place in an organized manner
* Announcement section: for all announcements related to study materials
